###### Basic HTML page with CSS and JS with comments

```html
<!DOCTYPE html>
<html>
    <head>
        <title>My First HTML Page</title>
        <!-- This is a comment -->
        <link rel="stylesheet" href="style.css">
    </head>
    <body>
        <h1>My First HTML Page</h1>
        <p>This is my first HTML page.</p>
        <!-- This is a comment -->
        <script src="script.js"></script>
    </body>
</html>
```
